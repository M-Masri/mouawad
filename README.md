# Mouawad

React + Vite + TypeScript + Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — start development server
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — run oxlint

## Structure

```
src/
  components/
    layout/
    sections/
    seo/
    ui/
  data/
  hooks/
  i18n/
  lib/
  pages/
  store/
  App.tsx
  fonts.css
  index.css
  main.tsx
```

Styling uses Tailwind CSS only (no custom CSS rules).
